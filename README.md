Bootcamp Santander 2025 – Fundamentos de Lógica de Programação
Introdução

Este repositório contém exercícios desenvolvidos em JavaScript durante o Bootcamp Santander 2025, com foco na prática de lógica de programação. O projeto está organizado em pastas, onde cada uma contém o enunciado do exercício e o respectivo código-fonte, facilitando o entendimento do problema e da solução implementada.

Exercícios
Classificador de Nível de Herói

Neste exercício foi desenvolvido um algoritmo que utiliza variáveis para armazenar o nome e a quantidade de experiência (XP) de um herói. A partir de estruturas condicionais, o programa classifica o nível do herói conforme o valor de XP e exibe a mensagem final no console.

Calculadora de Partidas Rankeadas

Neste exercício foi criada uma função que recebe como parâmetros o número de vitórias e derrotas de um jogador. Com base no saldo calculado, o programa utiliza estruturas de decisão para definir o nível correspondente e exibir o resultado no console.

Classes de um Jogo

Neste exercício foi implementada uma classe em JavaScript para representar um herói, com propriedades definidas no construtor e um método responsável pelo ataque. O tipo de ataque varia de acordo com o tipo do personagem, sendo definido por meio de estruturas condicionais.


Tecnologias utilizadas: JavaScript, Git, GitHub
