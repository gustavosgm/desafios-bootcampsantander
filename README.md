# Bootcamp Santander 2025 – Fundamentos de Lógica de Programação

---

## Introdução

Este repositório reúne projetos desenvolvidos em JavaScript durante o Bootcamp Santander 2025, com foco na prática dos fundamentos da lógica de programação. Os projetos estão organizados em pastas, contendo o enunciado e o respectivo código-fonte.

---

## Projetos

### 1 - Classificador de Nível de Herói

Programa que utiliza variáveis para armazenar o nome e a quantidade de experiência (XP) de um herói. Com base no valor de XP, o sistema classifica o nível do personagem e exibe o resultado no console.

---

### 2 - Calculadora de Partidas Rankeadas

Função que recebe o número de vitórias e derrotas de um jogador. A partir do saldo calculado (vitórias - derrotas), o programa define o nível correspondente e exibe o resultado no console.

---

### 3 - Classes de um Jogo

Implementação de uma classe para representar um herói, definindo suas propriedades e um método de ataque. O tipo de ataque varia conforme o tipo do personagem e é exibido no console.

---

## Métodos Utilizados

- **Projeto 1:** Variáveis, Operadores, Estruturas de decisões  
- **Projeto 2:** Variáveis, Operadores, Funções, Estruturas de decisões  
- **Projeto 3:** Classes e Objetos, Variáveis, Funções, Estruturas de decisões
